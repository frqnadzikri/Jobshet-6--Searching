1. Perbedaan:
tampilPosisi → menampilkan letak/index data
tampilDataSearch → menampilkan isi data mahasiswa
2. Fungsi break:
Untuk menghentikan perulangan saat data ditemukan agar tidak lanjut mencari
3. Fungsi pos:
Untuk menyimpan index hasil pencarian dan penanda apakah data ditemukan atau tidak
4. Jika data sama lebih dari satu:
Yang ditampilkan adalah data pertama yang ditemukan, karena loop langsung berhenti
5. Jika break dihapus:
Loop tetap berjalan sampai akhir, sehingga yang tersimpan adalah data terakhir yang sama dan program jadi kurang efisien
